# CFP Money – Controle Financeiro Pessoal

Aplicação web moderna para gerenciamento financeiro pessoal, com autenticação (Google/e-mail), dashboard interativo, categorias, metas, orçamentos, relatórios e exportação.

## Tecnologias
- React 18
- Vite
- TailwindCSS
- Firebase (Auth + Firestore)
- Chart.js / Recharts
- Framer Motion

## Deploy no GitHub Pages

1. Configure o Firebase e obtenha as credenciais.
2. Crie um arquivo `.env` com as variáveis conforme `.env.example`.
3. Execute:
   ```bash
   npm install
   npm run build
   npm run deploy
